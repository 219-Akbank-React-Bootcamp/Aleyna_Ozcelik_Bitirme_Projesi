# Patika & Akbank React Bootcamp Bitirme Projesi

### Kullanılan Teknolojiler:
```
> ReactJS
> Redux
> Axios
> Bootstrap 
> React-beautiful-dnd
> Create-react-app

```
### Özellikler:
```
> Kullanıcı Ekleme / Giriş Yapma
> Board Ekleme / Silme 
> Liste Ekleme /Silme / Güncelleme
> Kart Ekleme / Silme / Güncelleme

```
### Eksikler:
```
> Drag-Drop
> Kullanıcı Arası Paylaşım
> Yorum Yapma
> Label Ekleme 

```
